# Furnitech — план продукта

## Цель

Веб‑сайт «Фурнитех»: калькулятор для магазина с **админской** и **клиентской** частями.

- **Клиент**: калькулятор в духе типичного онлайн-конфигуратора (подбор профилей и комплектации).
- **Админ**: база материалов для калькулятора, заказы.

- **1С (задел)**: `external_id`, `last_synced_at` на сущностях; **артикул** (`article`) — для сопоставления с номенклатурой; детали в [ARCHITECTURE.md](ARCHITECTURE.md).
